# Foodlytics – Food Delivery SQL Analysis

## Project Overview
Foodlytics is a SQL-based analytics project that analyzes customer purchasing behavior, product performance, revenue generation, and Gold Membership effectiveness for a food delivery business.

## Objectives
- Analyze customer spending patterns
- Identify top-selling products
- Measure revenue performance
- Evaluate Gold Membership impact
- Generate business insights using SQL

## Database Tables
1. Product
2. Users
3. User_Name
4. GoldUsers_Signup
5. Sales

## ER Diagram
Users → Sales ← Product
Users → GoldUsers_Signup
Users → User_Name

- Microsoft SQL Server

## SQL Concepts Used
- Joins

- GROUP BY
- CTEs
- Window Functions

## Analysis Performed
- Revenue Analysis
- Customer Spending Analysis
- Product Popularity Analysis
- Membership Impact Analysis
- Customer Loyalty Analysis

## Key Findings
- Identified high-value customers
- Analyzed product performance
- Measured Gold Membership effectiveness
- Generated actionable business insights

## Project Outcome
This project demonstrates how SQL can be used to transform transactional data into meaningful business insights and support data-driven decision-making.
